Data Structures
Containers

Besides lists and tables, Mint supports a few more data structures.

Sets
Sets are lists that can only hold one of each unique element. For instance, you cannot put the number 3 in the same set twice.
someNumbers = Set{2.5, 2, 15, 15, 15, -32}
print someNumbers
The code above will print: Set{2.5, 2, 15, -32}
For efficiency reasons, sets may reorder their items in any way. Sets are most useful with the in operator, which returns true if an item is contained in the set, and false otherwise.
someNumbers = Set{2.5, 2, 15, 15, 15, -32}
print 2.5 in someNumbers    //Is 2.5 in the set? True.
print -33 in someNumbers    //Is -33 in the set? False.
print "frog" in someNumbers    //Is "frog" in the set? False.
print not (3.5 in someNumbers)    //Does the set not contain 3.5? True.
Sorted Lists
Sorted lists are lists that always maintain an ordering, from smallest to greatest, including when items are removed or added from the list after it is created.
srted = Sorted[10, 5, 2.5, 0, 1, 1.75]
print srted    //Prints Sorted[0, 1, 1.75, 2.5, 5, 10]
Matrices
A matrix is a 2-dimensional array of items used for various calculations such as matrix multiplication. A 1-dimensional matrix is considered a vector.
myMatrix = Matrix[0, 0, 1 | 0, 1, 0 | 1, 0, 0]
otherMatrix = Matrix[2, 5, 3 | 6, 7, 8 | 9, 10, 11]
product = myMatrix * otherMatrix
print product         //Prints Matrix[9, 10, 11 | 6, 7, 8 | 2, 5, 3]
vector = Matrix[60, 80.25, 101.9]
vector2 = Matrix[9, -18, -55]
dotProduct = vector.dot(vector2)
crossProduct = vector.cross(vector2)
print dotProduct      //Prints -6509.0
print crossProduct    //Prints Matrix[-2579.55, 4217.1, -1802.25]
Trees
Trees are data structures that have a root and more child trees. To make the tree:
  50
 /  \
2    25
Use:
myTree = Tree(50)
a = Tree(2)
b = Tree(25)
myTree.addChild(a)
myTree.addChild(b)
print myTree    //Prints Tree(50, Tree(2), Tree(25))
Trees can also be constructed in this manner:
familyTree = Tree("parents", Tree(["son", "wife"], Tree("grandchildren")), Tree(["daughter", "husband"], Tree("grandchildren")))
The second tree (familyTree) represents the following structure:
            parents
           /       \
 wife - son         daughter - husband
      |                      |
grandchildren          grandchildren
Which can be interpreted as two parents with a son and a daughter. Both of the parents' children are married, and they also have grandchildren.

Graphs
Graphs are collections of nodes and edges. A tree is a special type of a graph. A graph is constructed with the same syntax as a table, representing all the edges in the graph and each pair of nodes that each edge connects. For graphs, it is perfectly valid to have duplicate keys with different values. 
g = Graph{"A", "B" | "B", "C" | "C", "D" | "D", "E" | "D", "A" | "A", "C"}
Represents: 
     A
   / | \
  B  |  D - E
   \ | /
     C
Sorting Functions
Finally, there are subprograms that can sort lists, such as merge sort and quick sort.
import sort
print quickSort([9, 8, 7, 100, 200, -9])
Strings & Lists
Let us talk a bit more about regular strings and lists.

One interesting thing about lists...
If a subprogram takes in exactly one list as an argument,
you can omit the parentheses that call the subprogram:
import sort
quickSort[900, -4, 0, 1, 20, 28, 9, -450, 1000, 200, 256, 1024]
To remove items from a list, use subtraction.
femaleNames = ["Alice", "Jenny", "Jane", "Beth", "Sophia"]
femaleNames = femaleNames - ["Jane", "Alice"]
print femaleNames    // Prints ["Jenny", "Beth", "Sophia"]
To reverse a string, just make it a negative string by putting a negative sign in front of it.
someText = "indefinite integral"
backwards = -someText
print backwards
The above code will print the reversed string "largetni etinifedni".
The or operator on strings can be used to alphabetize strings.

The expression A or B will return A if A contains a string
that is alphabetically before B, and will return B otherwise.
/* If we alphabetize these words,
 * you get "what", "whence", "where", and "whoa"
 * The following statement prints "what", which is
 * the first word that comes alphabetically in this list
 * of 4 words.
 */
print "whence" or "where" or "what" or "whoa"
Strings can be compared by their lengths using the <, >, <=, and >= operators.
if thisString > thatString
    // This clause will only be executed if and only
    // if thisString has a greater length
    // than thatString.
end
import sort
// The following line will sort strings by length,
// and therefore it will display:
// ["tiny", "mouse", "tigress", "elephant"]
// onto the screen.
print quickSort["elephant", "tiny", "mouse", "tigress"]
