Syntax Highlighting
Mint source files are plaintext, so you can use whatever text editor you want.
However, you probably want syntax highlighting for Mint.
  • The Vim syntax file is being prepared (not finished yet).
  • The Notepad++ syntax file is finished. It is named Mint.xml and is located in the root of the directory where you unzipped the Mint distribution. Click here if you have trouble installing it.

Running Mint Programs
These steps apply on any OS.
From the terminal, use java -jar Mint4.0.jar to execute the default file program.mint
To run any file, use java -jar Mint4.0.jar NAME_OF_YOUR_FILE.mint

Java must be installed and aliased as java.

Read-Eval-Print-Loop
Mint comes with a REPL, similar to the ones that come with Python or Ruby. By default, program.mint contains some code that runs the REPL, but you are free to replace it with your own Mint program.

Code Examples
The Hello World program consists of the following line: 
print "Hello world!"

In Mint, operators go between their operands, like most languages.
When you divide two integers, a double-precision float is produced. String concatenation uses the + sign. 
print 300
print 36 * (8 + 14)
print "My name is " + "Alice."
a = 81 / 4
print a    // a is 20.25, a double-precision float
b = a + 87
print b
(if you are using the REPL, you may omit the print keyword since evaluated expressions are printed automatically)

To perform integer division, use intDiv(a, b). intDiv will compile into a single bytecode instruction.
To create a rational number, known as a "ratio" in Mint terminology, use the colon (:). 
twenty = intDiv(81, 4)
threeOverFour = 3:4
ten = twenty:2
print ten * threeOverFour    // Prints 15:2 (meaning the fraction 15/2)

To read from the console, use the keyword input. show is the same as print, but does not produce a newline at the end. 
show "Enter your name:"
yourName = input
print "Your name is: " + yourName

Mint has the following single-line and multi-line comments:
// This is a line comment.

/* This is
a block comment */

Finally, here is what the REPL might look like:
Mint 4.0> text = "This is a string!"
Mint 4.0> text.upper()
"THIS IS A STRING!"
Mint 4.0> -37.next()
-36
Mint 4.0> b = -37.bytes()
ByteArray[255, 255, 255, 219]
Mint 4.0> b.type()
#ByteArray
Mint 4.0> h = toHex.map(b)
["0xff", "0xff", "0xff", "0xdb"]
Mint 4.0> h.type()
#List
Mint 4.0> result = ""
""
Mint 4.0> for i = 0; i < h.length(); i++
Mint 4.0*     result += h[i].remove("0x")
Mint 4.0* end
Mint 4.0> "0x" + result.upper()
"0xFFFFFFDB"
