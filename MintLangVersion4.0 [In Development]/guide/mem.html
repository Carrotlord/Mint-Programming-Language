Variable Deletion
Creating and Erasing Variables

You already know how to create variables. Use the = operator.
To delete variables, use erase, followed by the names of one or more variables. Separate the names with commas.
x = 5
y = [20, 30]
z = 300
erase x, y
print z
print y     // Error: y is not defined.
print x     // Error: x is not defined.
You probably won't be using the erase keyword very often, since variables are usually local to their containing functions.
However, there is one other useful variable manipulator that will swap the internal values of 2 variables:
flip a, b   // Switches the values of a and b.
You might notice that flip x, y is the same as: 
temporary = x
x = y
y = temporary
But looks far cleaner.
